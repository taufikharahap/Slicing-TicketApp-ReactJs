## 🛠️ Installation Steps

1. Clone the repository

```bash
git clone https://github.com/taufikharahap/TickitzApp-React-Tailwind.git
```

2. Install dependencies

```bash
npm install
```

3. Run the app

```bash
npm run dev
```

🌟 You are all set!

-   JavaScript
-   React Js
-   React-Redux
-   Tailwind
